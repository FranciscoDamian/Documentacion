BANCOMEX

Description

  • What was out motivation at the time of creating this project? We wanted to create our version of this bank while putting in practice all the things that we have learn in the bootcamp.
  • What problem does it solve? It's a solution for a small bank
  • What did you learn? How to work as a team Node JS, Express, Sequelize, PostgreSQL, express-validator

Installation

1. Clone our project

Open a terminal and the following command:

$ git clone https://github.com/FranciscoDamian/bancomex_backend.git

2. Install dependencies

Install PostgreSQL

https://www.postgresql.org/download/

Open a terminal and the following command:

cd bancomex_backend
npm install

3. Migrate DB

    npm install sequelize-cli
    npm install sequelize
    
    sequelize db:migrate
    sequelize db:seed:all

Usage

1. Run our project

nodemon

2. Consume our endpoints using Postman, ThunderClient, etc

POST localhost:3001/api/v1/login
